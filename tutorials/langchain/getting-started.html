Getting Started with LangChain

Install LangChain, understand its core concepts, and build your first chain.

What is LangChain?

LangChain is a framework for building applications with large language models. It provides tools for chaining LLM calls, managing memory, using tools, and more.

Installation

pip install langchain langchain-openai

Your First Chain

from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate

# Initialize the model
llm = ChatOpenAI(model="gpt-4o-mini")

# Create a prompt template
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("user", "{input}")
])

# Create the chain
chain = prompt | llm

# Run it
response = chain.invoke({"input": "Hello!"})
print(response.content)
💡 LCEL (LangChain Expression Language)

The | operator is LCEL syntax. It chains components together: prompt | llm | parser
