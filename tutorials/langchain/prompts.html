Prompt Templates

Create reusable, dynamic prompts with variables and formatting.

Basic Prompt Templates

from langchain_core.prompts import PromptTemplate

# Simple template with variables
template = PromptTemplate(
    input_variables=["topic"],
    template="Explain {topic} in simple terms."
)

prompt = template.format(topic="quantum computing")
print(prompt)  # "Explain quantum computing in simple terms."

Chat Prompt Templates

from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are an expert in {domain}."),
    ("user", "{question}")
])

messages = prompt.format_messages(
    domain="Python",
    question="What are decorators?"
)
💡 Prompt Best Practices

Keep templates focused on one task. Use descriptive variable names. Version control your prompts!
