Gemini Embeddings API

Generate text embeddings with Gemini, build semantic search, and use embeddings in RAG pipelines.

Generating Embeddings with Gemini

Gemini provides the text-embedding-004 model for generating high-quality text embeddings. With 768 dimensions by default and optional dimensionality reduction, it's ideal for semantic search, clustering, and RAG.

Python - generate embeddings
import google.generativeai as genai
import os

genai.configure(api_key=os.environ["GOOGLE_API_KEY"])

# Single embedding
result = genai.embed_content(
    model="models/text-embedding-004",
    content="Python is a great programming language for AI"
)
print(f"Dimensions: {len(result['embedding'])}")  # 768

# Batch embeddings (more efficient)
texts = [
    "How do I build a RAG system?",
    "What is LangChain?",
    "Explain vector databases",
]
result = genai.embed_content(
    model="models/text-embedding-004",
    content=texts
)
for i, emb in enumerate(result["embedding"]):
    print(f"{texts[i][:40]}: {len(emb)} dims")

Task Types for Better Accuracy

Gemini embeddings support task types that optimize the embedding for different use cases.

Python - task-optimized embeddings
# Use task_type for best accuracy per use case
TASK_TYPES = {
    "retrieval_document": "For indexing documents into a vector store",
    "retrieval_query": "For encoding search queries",
    "semantic_similarity": "For comparing sentence similarity",
    "classification": "For text classification tasks",
    "clustering": "For clustering documents",
}

# Index documents with retrieval_document task
doc_embedding = genai.embed_content(
    model="models/text-embedding-004",
    content="FastAPI is a modern Python web framework",
    task_type="retrieval_document"
)["embedding"]

# Query with retrieval_query task
query_embedding = genai.embed_content(
    model="models/text-embedding-004",
    content="how to build an API in Python?",
    task_type="retrieval_query"
)["embedding"]

Semantic Search with FAISS

Python - semantic search engine
import numpy as np
import faiss

docs = [
    "FastAPI builds REST APIs with Python type hints",
    "LangChain connects LLMs with tools and memory",
    "Ollama runs LLMs locally on your machine",
    "ChromaDB is a vector database for AI apps",
]

# Embed all documents
result = genai.embed_content(
    model="models/text-embedding-004",
    content=docs, task_type="retrieval_document"
)
vectors = np.array(result["embedding"], dtype="float32")
faiss.normalize_L2(vectors)

index = faiss.IndexFlatIP(vectors.shape[1])
index.add(vectors)

# Search
query = genai.embed_content(
    model="models/text-embedding-004",
    content="What tool runs AI models offline?",
    task_type="retrieval_query"
)["embedding"]
q = np.array([query], dtype="float32"); faiss.normalize_L2(q)
scores, idx = index.search(q, k=2)
for score, i in zip(scores[0], idx[0]):
    print(f"[{score:.3f}] {docs[i]}")
💡 Dimensionality reduction

Set output_dimensionality=256 in embed_content to reduce from 768 to 256 dimensions — 3x faster retrieval with minimal accuracy loss for most use cases.
