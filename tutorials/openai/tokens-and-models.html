Tokens & Model Selection

Learn how tokens work, compare GPT models, and choose the right model for your use case.

Understanding Tokens

Tokens are the basic units that AI models use to process text. Think of them as pieces of words - on average, one token equals about 4 characters or 3/4 of a word in English.

# Example token counts:
"Hello"         # 1 token
"Hello world"   # 2 tokens
"ChatGPT"       # 2 tokens (Chat + GPT)
"extraordinary" # 1 token (common word)

Counting Tokens with tiktoken

OpenAI provides the tiktoken library to count tokens before making API calls:

import tiktoken

# Get the encoder for GPT-4
encoder = tiktoken.encoding_for_model("gpt-4o")

text = "Hello, how are you today?"
tokens = encoder.encode(text)

print(len(tokens))  # Number of tokens
print(tokens)       # Token IDs

Model Comparison

OpenAI offers several models with different capabilities:

  • GPT-4o - Most capable, best for complex tasks, supports vision
  • GPT-4o-mini - Fast and cost-effective, great for most tasks
  • GPT-4 Turbo - Large context window (128K tokens)
  • GPT-3.5 Turbo - Legacy model, still fast and cheap
💡 Choosing a Model

Start with GPT-4o-mini for development and testing. Only upgrade to GPT-4o when you need its advanced reasoning or vision capabilities.
