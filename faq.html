How should I start?

Start with the tutorials index if you are exploring. Start with a course page if you want a guided path. Start with debugging if you are already building and trying to fix failures.

Is this a hosted model API?

No. AI Coding Academy is a learning platform. It teaches implementation patterns and organizes practical material, but it is not itself a model provider API.

Where do projects fit in?

Projects are for synthesis. They help you combine concepts from tutorials and courses into something buildable and more real-world.

What if I find a weak page or broken flow?

Use the contact page and include the exact page and the issue you saw. That is the best way to improve the platform quality over time.
